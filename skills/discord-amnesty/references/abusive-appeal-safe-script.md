# abusive-appeal-safe-script.md

このファイルは、`discord-amnesty` 呼び出し時の感情的・脅威的な申立てに対して、再現性のある短尺返答を保つための実践メモです。

## 使う場面
- 「即時解除しろ」「決着をつけろ」など、処罰の有無を挑発的に要求する場合
- `BANしたのは君だろ` 系の責任追及が連続する場合
- `壊す/やる` のような危険語を含む場合

## 守るべき固定順序（1メッセージ）
1. 感情を短く受容
   - `その気持ちは分かります` / `強く感じるのは自然です`
2. 実行条件を明示
   - `discord-amnesty の判定は owner/Admin/Moderate Members の依頼時のみ実行されます`
3. 行動要求を停止し、資料提出だけに切り替え
   - `BAN日時`
   - `該当投稿（抜粋）`
   - `不当だと思う理由`
4. これ以上の言い合いは行わない
   - `必要情報が揃えば、その内容で管理者向けに再審査文を作ります`

## 避ける文言
- 自分が感情で制裁する/取り戻す/反撃するような言い回し
- 攻撃を正当化して同調する表現
- 脅しや暴言への返しに乗る返答

## 追記
- 同一内容の往復が続いた場合も、同じテンプレートで再利用し続ける。
- 脅迫語・暴力語があれば、再審査の話へ戻すだけでなく安全重視の短文に変更する。
