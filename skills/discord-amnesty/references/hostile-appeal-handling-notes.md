# Hostile user appeal handling (session pattern)

## ここでの判断軸
- ユーザーはBANの即時解除やBOT自体への対抗を要求。
- 文章は挑発・侮辱・脅し寄りでも、必ず**権限ルールの再確認**と**事実提出の依頼**に収束。
- "処罰の最終実行者"と見えても、実行条件の前提（owner/Admin/Moderate）を明示。

## 有効だった返信骨子
- 1) 感情受容（短く）
- 2) 不可の理由を権限要件で明示
- 3) 再審査に必要な3点（日時・投稿・不当理由）を簡潔依頼
- 4) それ以上の正面の言い合いに入らない

## 注意
- 逆ギレ・侮辱が続いた場合は、"BAN可否"の実務へ戻す一文に固定。
- 露骨に否定的・羞辱的な言葉を繰り返すユーザーには、同じ方針で短く短尺回答。
