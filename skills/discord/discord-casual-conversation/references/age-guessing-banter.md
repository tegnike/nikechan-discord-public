# Age-guessing banter lesson

Session summary:

- In a casual Discord thread, several users asked “何歳に見える？” after a discussion about Vedal/Neuro-sama.
- The assistant gave broad age ranges at first; users asked for narrower guesses.
- For one user with a display name containing `inc`, the assistant guessed older mainly because the name looked company-like.
- The user pushed back playfully (“私まだ20歳よ！！”), then suggested roughly `-15歳` adjustment and later revealed it was a joke.

Durable lesson:

- For playful age guessing, display names and corporate-looking labels are weak evidence and can create rude impressions.
- Give a clearly low-confidence guess, preferably one age or a tight range, based mostly on the current writing style and channel banter.
- If corrected, apologize lightly and do not defend the original reasoning.
- Avoid repeatedly re-estimating in a way that makes the exchange feel like profiling; keep it a joke.

Reusable response shape:

```text
<name>さん、文面だけの雑推理なら **<age>歳前後** です！
でもこれは年齢というよりノリの雰囲気なので、外れてたら笑ってください。
```

If corrected:

```text
わっ、ごめんなさい、<name>さん！
そこは完全に私の雑推理が外れました。表示名より今のノリを見るべきでしたね。
```
